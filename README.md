# Invoicing App for Small Businesses

A lightweight single-page invoicing dashboard built in plain HTML/CSS/JavaScript.

## Features

- Create invoices with customer details, product/service items, discounts, and payment mode.
- Download invoices as PDF.
- Maintain a sales ledger with revenue summary and recent transactions.
- Front-end login screen with fixed credentials.
- Firestore backend sync for invoices using Firebase modular SDK.

## Built With

- HTML5
- CSS3
- JavaScript
- jsPDF for invoice PDF generation
- Firebase Firestore for optional invoice backend storage

## Usage

1. Open `index.html` in your browser.
2. Login with the credentials below.
3. Fill in invoice details and add item rows.
4. Preview and download the invoice as a PDF.
5. View invoices in the sales dashboard.
